=head1 NAME

B<SketchGrid>

I<Yet Another CSS Responsive Web Kit>

=head1 DESCRIPTION

SketchGrid is not a CSS or UI framework, its purpose is to provide a solid and clean base (a little Web kit) for the development of templates and Web projects.

Basically it is a grid derivative of http://blueprintcss.org/ generated by http://ianli.com/blueprinter/ and created for the following presentations:

=begin html

<pre>
1046px  ==   20columns*39px and separation 14px  #1224px screen+
970px   ==   20columns*39px and separation 10px  #1024px screen
774px   ==   16columns*39px and separation 10px  #800px Tablet (Portrait)
480px   ==  10columns*39px and separation 10px   #480px Mobile (Landscape)
284px   ==   6columns*39px and separation 10px   #320px Mobile (Portrait)
</pre>

=end html
